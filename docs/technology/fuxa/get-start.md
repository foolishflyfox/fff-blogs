# 开始使用 FUXA

本教程旨在演示FUXA最常用功能的基本操作方法。您将学习如何：

- 将设备连接至FUXA
- 将设备数据推送至FUXA
- 构建实时图形界面的SCADA/HMI/仪表板
- 设定阈值并触发报警机制

## 环境准备

您需要确保FUXA服务器已启动并处于运行状态。请参照[安装指南](./install-and-run.md)完成FUXA的安装。安装完成后，可通过以下网址访问FUXA用户界面：http://localhost:1881 。

FUXA网页服务器主要提供两个功能页面：

- 终端用户可视化界面：可通过 http://localhost:1881/home 访问
- 项目设计与编辑器：可通过 http://localhost:1881/editor 访问
