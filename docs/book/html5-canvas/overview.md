# HTML5 Canvas 核心技术 - 图形、动画与游戏开发

知识要点思维导图：

```mindmap
# Canvas 技术
## canvas 标签
### 属性
- `width` 画布逻辑宽
- `height` 画布逻辑高
### 方法
- `getContext` 获取绘图环境
- `toDataURL` 生成 Base64 URL
- `toBlob` 生成类文件 Blob 对象
## context 绘图环境
### 属性
- `font`: 字体
- `fillStyle`: 填充色
- `strokeStyle`: 描边色
### 方法
- `fillText`: 填充文本
- `strokeText`: 描边文本
```
