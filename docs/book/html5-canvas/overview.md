# HTML5 Canvas 核心技术 - 图形、动画与游戏开发

知识要点思维导图：

```mindmap
# Canvas 技术
## canvas 标签
### 属性
- `width` 画布逻辑宽
- `height` 画布逻辑高
### 方法
- `getContext` 获取绘图环境
- `toDataURL` 生成 Base64 URL
- `toBlob` 生成类文件 Blob 对象
## context 绘图环境
### 属性
- `canvas`: 所属 canvas 对象
- `fillStyle`: 填充色
- `font`: 字体
- `globalAlpha`: 全局透明度
- `globalCompsiteOperation`: 全局叠加模式
- `lineCap`: 线段端点形状
- `lineWidth`: 线段宽度
- `lineJoin`: 线段拐点形状
- `miterLimit`: 最大斜接长度
- `shadowBlur`: 阴影模糊级数
- `shadowColor`: 阴影颜色
- `shadowOffsetX`: 阴影 x 偏移
- `shadowOffsetY`: 阴影 y 偏移
- `strokeStyle`: 描边色
- `textAlign`: 文本水平对齐方式
- `textBaseline`: 文本垂直对齐方式
### 方法
- `fillText`: 填充文本
- `strokeText`: 描边文本
```
